# List of Reserved Names
 
If a proto field has a name in the following keyword list, it will be replaced by the name
followed by a quote (i.e. the proto field name `type` will generate `#type'`).
 
* `case`
* `class`
* `data`
* `default`
* `deriving`
* `do`
* `else`
* `foreign`
* `if`
* `import`
* `in`
* `infix`
* `infixl`
* `infixr`
* `instance`
* `let`
* `module`
* `newtype`
* `of`
* `then`
* `type`
* `where`
* `mdo`
* `rec`
* `pattern`
* `proc`
